💾 Auto-Save Status
Checking…

Auto-saves to this browser only. Clearing Safari cache or switching devices loses data. Use "Export orbit-data.json" below — upload that file to Netlify after each session for permanent cross-device persistence.

🔒 Persistent Data — orbit-data.json

After each editing session: tap Export Data → download saves as orbit-data.json → drag that file into Netlify alongside index.html. App loads it automatically on every visit, every device. Deletions are remembered because the whole list replaces the old one.

⬆ GitHub Direct Sync (one-tap save)

Write orbit-data.json directly to your GitHub repo without manual upload. Requires a GitHub Personal Access Token with repo scope.

One-time setup: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → check repo → copy token.

☁ Media Offload — Free Storage from Local

When localStorage fills up it's usually base64 images/videos. This scans your entries, finds local media, and uploads it to GitHub or Cloudinary — then replaces the local data with the cloud URL. Credentials stored in your browser only.

Uses your existing GitHub token above. Media uploads to a media/ folder in the same repo. One-time: set owner/repo/token in the GitHub section above.
Free 25GB at cloudinary.com. Setup: Settings → Upload → Upload presets → Add preset → set Signing mode: Unsigned → save. Copy the Cloud Name and Preset Name here.

🗂 Meme Folder → Netlify (No base64)

The right way to store meme images at scale. Images stay as files — no bloated base64 in your HTML.
Step 1: Organize memes into folders by category (folder name = tag).
Step 2: Run the Node.js scanner below — it generates a memes-import.json.
Step 3: Upload your meme folders + the HTML to Netlify (drag the whole folder).
Step 4: Import the JSON here — images load from their Netlify URLs.

// Save as scan-memes.js and run: node scan-memes.js // Put this file next to your meme folders, e.g.: // my-site/ // index.html // orbit-data.json // scan-memes.js ← this script // memes/ // dark/ ← folder name becomes a tag // meme1.jpg // meme2.png // absurd/ // whatever.gif // wordplay/ // pun.jpg const fs=require('fs'),path=require('path'); const MEMES_DIR='./memes'; // relative to this script const NETLIFY_BASE='/memes'; // URL path on your Netlify site const IMG_EXT=/\.(jpg|jpeg|png|gif|webp|avif|svg)$/i; const tags={};let id=Date.now();const jokes=[]; fs.readdirSync(MEMES_DIR).forEach(folder=>{ const folderPath=path.join(MEMES_DIR,folder); if(!fs.statSync(folderPath).isDirectory())return; const tagId=folder.toLowerCase().replace(/[^a-z0-9]/g,'_'); tags[tagId]={id:tagId,name:folder,color:null,gid:null}; fs.readdirSync(folderPath).filter(f=>IMG_EXT.test(f)).forEach(file=>{ jokes.push({ id:id++,type:'meme', body:file.replace(/\.[^.]+$/,'').replace(/[-_]/g,' '), punch:'',tags:[tagId], md:null,mt:null, mu:`${NETLIFY_BASE}/${folder}/${file}`, mu_is_image:true, image:null,variants:[],links:[] }); }); }); const out={ tags:Object.values(tags), jokes, _note:`${jokes.length} memes from ${Object.keys(tags).length} folders. Upload your /memes/ folder to Netlify alongside your HTML.` }; fs.writeFileSync('memes-import.json',JSON.stringify(out,null,2)); console.log(out._note);
After importing the JSON, export orbit-data.json and upload it alongside your /memes/ folder to Netlify.
